# kmskeyalias

This project creates:
- KMS key with alias
    - Three policies:
        - Allow root access of the key
        - Allow administration of the key
        - Allow use of the key
- Asymmetric key with alias
    - Three policies:
        - Allow root access of the key
        - Allow administration of the key
        - Allow use of the key
